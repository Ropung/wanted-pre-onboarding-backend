## 서비스 개요

- 본 서비스는 기업의 채용을 위한 웹 서비스 입니다.
- 회사는 채용공고를 생성하고, 이에 사용자는 지원합니다.


## 맡은 기능 및 환경

- **채용공고 서비스**(등록, 수정, 삭제, 목록 조회, 검색 조회)
- 채용 조회(+회사가 올린 다른 채용공고)
- 도커 컴포즈 **컨테이너 환경**구축
- **Flyway** 적용


## 필수 기술요건

- ORM 사용하여 구현.
- RDBMS 사용 (SQLite, PostgreSQL,MySql 등).
